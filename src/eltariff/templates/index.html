Detta verktyg hjälper svenska elnätsbolag att skapa ett maskinläsbart API för sina tariffer enligt RISE Eltariff API-standarden. Ladda upp din tariffbeskrivning, så genererar AI:n ett komplett deployment-paket med Docker.

Rate limit: 10 AI-analyser per timme.

Vad är RISE Eltariff API-standarden?

RISE Eltariff API är en öppen standard utvecklad av RISE (Research Institutes of Sweden) tillsammans med svenska elnätsbolag för att göra elnätstariffer maskinläsbara.

Varför behövs detta? Idag publicerar de flesta elnätsbolag sina tariffer som PDF:er eller webbsidor som är svåra för appar och tjänster att tolka automatiskt. Med ett standardiserat API kan:

  • Energiappar visa exakta kostnader för dina kunder
  • Smarta hem optimera förbrukning baserat på tariffen
  • Elbilsladdning schemaläggas när det är billigast
  • Konsumenter jämföra elnätskostnader enkelt

Hur fungerar standarden? API:et beskriver tre typer av avgifter:

  • fixedPrice - Fasta avgifter (abonnemang, säkringsavgift)
  • energyPrice - Energiavgifter per kWh (ofta tidsdifferentierade)
  • powerPrice - Effektavgifter per kW (baserat på förbrukningstoppar)

Varje avgift kan ha tidsregler (höglast/låglast, sommar/vinter) och beräkningsregler (t.ex. snitt av 3 högsta effekttoppar).
